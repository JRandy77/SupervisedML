# MiCM Workshop: Introduction to Supervised Machine Learning (Tabular, Medical)

## Objectives
- Understand the supervised learning workflow on medical tabular data.
- Practice regression, classification, regularization, nonlinearity, and MLPs.
- Build and evaluate full pipelines with preprocessing.
- Apply all concepts to a new medical dataset in a short hackathon.

## Schedule (9:00 to 3:15)
- 9:00–9:30 Introduction and context for computational medicine
- 9:30–10:15 Linear models on medical data
- 10:15–11:00 Regularization for high dimensional data
- 11:00–11:15 Break
- 11:15–12:00 Nonlinearity and feature engineering
- 12:00–12:30 MLPs for tabular data
- 12:30–1:15 Lunch
- 1:15–2:00 Training, metrics, hyperparameter search
- 2:00–3:00 Mini hackathon
- 3:00–3:15 Presentations and wrap up
